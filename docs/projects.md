### Project November

**Status:** In Development &nbsp;·&nbsp; **Type:** Personal Product

A personal productivity and self-improvement platform focused on tracking habits, fitness, workouts, sleep, and long-term personal growth through a clean dashboard experience.

**The Problem**

Most productivity tools treat each dimension of personal improvement in isolation. Habit trackers don't connect to fitness goals. Sleep data sits in a separate app. There's no unified view of personal growth over time.

**The Approach**

Project November is a single dashboard that treats personal productivity as a system. Track habits, log workouts, monitor sleep, and measure long-term progress — all in one place, with a clean interface that encourages daily use.

**Tech Stack**

| Layer | Technologies |
|:---|:---|
| Frontend | Next.js, TypeScript, Tailwind CSS |
| Backend | Firebase |
| Design Philosophy | Minimal, dashboard-first, data-driven |

### Command Center

**Status:** In Development &nbsp;·&nbsp; **Type:** Internal Tool / Product

A centralized dashboard designed to manage projects, workflows, automation, and business operations with a productivity-first interface.

**The Problem**

Managing multiple projects, workflows, and business operations across scattered tools creates friction and reduces visibility. Context switching between tools wastes time that should be spent building.

**The Approach**

A single command center that consolidates project management, workflow automation, and operational monitoring into one customizable interface.

**Tech Stack**

| Layer | Technologies |
|:---|:---|
| Frontend | Next.js, Tailwind CSS, shadcn/ui |
| Backend | Firebase, Supabase |
